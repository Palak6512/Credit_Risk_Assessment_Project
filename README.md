# Credit Risk Assessment Using Machine Learning and Deep Learning

This project predicts customer credit default risk using both Machine Learning and Deep Learning models.

## Models Used
- Logistic Regression
- Random Forest
- XGBoost
- ANN
- CNN
- RNN
- LSTM

## Datasets Used
- UCI Credit Card Dataset
- German Credit Dataset

## How to Run
1. Download the datasets and code files.
2. Open the notebook file in Jupyter Notebook or Google Colab.
3. Install required libraries such as pandas, numpy, scikit-learn, tensorflow, xgboost, and imbalanced-learn.
4. Run the notebook cells step by step.
